# 📋 HTML Nested Lists Assignment

This repository contains three HTML nested list tasks created using **HTML only**. The project demonstrates proper nesting, indentation, and different list types while following HTML best practices.

## 📌 Project Overview

The project includes:

- ✅ Task 1 - Nested Ordered Lists
- ✅ Task 2 - Nested Unordered Lists
- ✅ Task 3 - Mixed Nested Lists (Ordered + Unordered)

Each task is implemented in a separate HTML file for better organization and readability.
